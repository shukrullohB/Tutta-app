# Tutta

Tutta is a mobile rental marketplace app for short-term stays, where users can rent a home or a room from **1 day up to 1 month**.

The platform also includes a **skill-exchange / free stay** model, where hosts can accept guests for free in exchange for useful skills such as speaking English, content creation, design, or other help. This feature is planned as a **premium host option**.

## Project Vision

Tutta is designed for:
- short-term home and room rentals
- daily, weekly, and monthly pricing
- easy booking experience
- safe communication between guest and host
- verified listings and users
- premium features for special host needs

## Core Features

### Guest Features
- Sign up / Login
- Search homes and rooms
- Filter by price, location, dates, and type
- View listings on map
- View listing details
- Select booking dates
- Book a listing
- Pay through local payment systems
- Chat with host
- Leave reviews

### Host Features
- Sign up / Login
- Create and manage listings
- Upload photos and videos
- Add location and property details
- Set daily, weekly, and monthly prices
- Accept or manage bookings
- Chat with guests
- Receive reviews

### Premium Features
- Skill-exchange / free stay listings
- Premium-only host tools
- Better listing visibility in future versions
