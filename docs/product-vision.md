# Tutta – Product Vision

## Overview
Tutta is a mobile marketplace where people can rent homes or rooms for short stays.

Guests can book places from **1 day up to 1 month**.  
The platform focuses on **simple, fast, and affordable** short-term stays.

## Problem
Many people need temporary accommodation, for example:
- travelers
- students
- people moving between cities
- temporary workers

Existing solutions are often **too expensive**, **too complicated**, or **not flexible** for short stays.

## Solution
Tutta enables people to:
- list a room or home for short-term rent
- set **daily, weekly, or monthly** pricing
- communicate directly between guest and host
- request and book stays easily through a mobile app

## Differentiator (Premium Feature): Skill-Exchange Stays
Tutta will introduce a **skill-exchange stay model**.

Premium hosts can offer **free or discounted accommodation** if the guest can provide useful skills such as:
- English speaking / tutoring
- design
- programming
- content creation

This is intended to increase value for hosts beyond money and create a unique marketplace angle.

## MVP Goal
Launch a working MVP that supports:
- listing homes/rooms
- booking short stays (1 day–1 month)
- chat between guest and host
- basic payments

The focus for the first release is **speed, simplicity, and reliability**.
