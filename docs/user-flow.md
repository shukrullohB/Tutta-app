# Tutta – User Flow

## Guest Flow

1. User opens the app
2. User signs up or logs in
3. User searches for homes or rooms
4. User applies filters
5. User opens listing details
6. User selects booking dates
7. User confirms booking
8. User makes payment
9. User chats with host
10. User stays in the place
11. User leaves review

---

## Host Flow

1. Host registers in the app
2. Host creates listing
3. Host uploads photos
4. Host sets price
5. Host sets availability
6. Host receives booking request
7. Host communicates with guest
8. Host receives payment
9. Host receives review

---

## Premium Host Flow

1. Host subscribes to premium
2. Host creates skill-exchange listing
3. Host specifies required skill
4. Guest applies for free stay
5. Host accepts guest
